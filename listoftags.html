You can use the following tags in your formats:
{name} - Name of the item
{type} - Type of item (Wordpress Plugin or Free Wordpress Theme)
{url} - URL to download the latest version from wordpress extend
{version} - latest version number of the item
{lastupdate} - Date of last update of the item files
{today} - No. of downloads today (based on saved data)
{yesterday} - No. of downloads yesterday (based on saved data)
{lastweek} - No. of downloads since last week (based on saved data)
{total} - Total no. of downloads (based on saved data)
{lastsync} - Date of last synchronize
{freshness} - Time difference between last synchronize and now
{dateadded} - Date when the data was first loaded, NOT when you upload it to wordpress extend
